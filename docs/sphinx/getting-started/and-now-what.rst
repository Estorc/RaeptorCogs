And now what?
=============
Now that you have successfully built and run your first RæptorCogs application, you may be wondering what to do next.
Here are some suggestions on how to continue your journey with RæptorCogs:

Explore the Documentation
--------------------------
The RæptorCogs documentation is a comprehensive resource that covers all aspects of the framework.
You can find detailed explanations of classes, functions, and features, as well as tutorials and examples.
Start by exploring the :doc:`Tutorials <../tutorials/introduction>` section to learn more.

Experiment with Examples
-------------------------
The RæptorCogs repository includes a variety of example applications that demonstrate different features and
capabilities of the framework. You can find these examples in the `examples` directory of the source code.
Try running and modifying these examples to see how they work and to get a feel for how to use RæptorCogs
in your own projects.
